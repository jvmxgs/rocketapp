# rocketapp

### instalar

```
git clone https://github.com/jvmxgs/rocketapp.git

cd rocketapp/build

npm install

cd www 

npm install
```


### Enpaquetar:

En la raiz del proyecto

```
webpack-dev-server
```


### Correr la app:

 En una segunda terminal

```
cd build

cordova platform add browser

cordova run browser --live-sync
```
